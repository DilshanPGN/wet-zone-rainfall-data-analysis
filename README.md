# wet-zone-rainfall-data-analysis
Analyzation of wet zone data of Sri Lanka past 30 years

## ITA Trend Analysis App

Run the Streamlit UI for Innovative Trend Analysis (station selection, date range, time interval, export PNG/PDF, AI interpretation):

```bash
pip install -r requirements.txt
streamlit run app_trend_analysis.py
```

Then open the URL shown in the terminal (e.g. http://localhost:8501).
